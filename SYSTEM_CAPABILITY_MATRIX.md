# System Capability Matrix: AHH WFM

This document outlines the capabilities, implementation status, schema definitions, endpoints, and verification checks for the **AHH WFM** system.

| Module | Status | Database | API | Web UI | Mobile UI | Tested |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| **Employee Directory** | Completed | `Employee`, `Department` | `/api/v1/employees`, `/api/v1/departments` | Workforce list grid, Add/Edit Employee form, Department selector | Read-only profile view, credentials, active roles | Unit tested CRUD, soft delete, status switches |
| **Attendance & Geofencing** | Completed | `AttendanceRecord`, `Worksite`, `AttendanceCorrection` | `/api/v1/attendance/*`, `/api/v1/worksites` | Geolocation worksites map pins, clock-in logs, correction reviews | Shift timers, check-in/out, GPS mock location changer, corrections requests | Haversine formula calculation, double clock-in blocking, geofencing limits |
| **Leave Ledger Engine** | Completed | `LeaveType`, `LeaveBalance`, `LeaveBalanceLedger`, `Holiday` | `/api/v1/leave-balances`, `/api/v1/leave-types`, `/api/v1/holidays` | Admin manual balances adjustments, carry-over rules config, holidays calendar list | Allotment meters, leave requests, working days calculation (Fri/Sat aware) | Working days logic, holiday exemptions, ledger transactions audits |
| **Multi-Level Approvals & SLA** | Completed | `LeaveApprovalWorkflow`, `LeaveApprovalStep`, `LeaveApprovalHistory`, `LeaveApprovalDelegation` | `/api/v1/leaves/approve`, `/api/v1/leaves/reject`, `/api/v1/leaves/history`, `/api/v1/approval-workflows`, `/api/v1/approval-delegations` | Approvals inbox split by Direct/Delegated/Escalated queues, delegation configurations, audit history timelines | Approval stepper progress checklist (e.g. Supervisor -> Manager -> HR), status indicators | Auto-approvals threshold, escalation cron trigger (72h limit), delegation transfers |
| **Shift Scheduling & Rotations** | Completed | `ShiftTemplate`, `RotationTemplate`, `ShiftAssignment` | `/api/v1/shifts/templates`, `/api/v1/shifts/assignments`, `/api/v1/shifts/assignments/bulk`, `/api/v1/shifts/assignments/rotations`, `/api/v1/shifts/rotations` | Grid timeline roster board, templates config cards, bulk assigner drawer, rotation applicator forms | Personal roster calendar, My shifts details view, upcoming shifts list | Overlapping shifts warning, approved leaves clash checks, inactive employee blocks |
| **Overtime & Swaps (Phase 4B)** | Completed | `ShiftSwapRequest`, `OvertimeRate`, `AttendanceRecord` (extensions) | `/api/v1/shifts/swaps`, `/api/v1/shifts/coverage`, `/api/v1/shifts/overtime`, `/api/v1/overtime-rates` | Coverage heatmap toggle overlays (Red/Green/Yellow), Drag-and-drop rosters, swap approvals, overtime reviews, rates panel | Bilateral swap request form, peer list selectors, overtime status tracker & pay display | Checkout OT calculations, QAR multipliers, atomic swap executions, leave date overrides |
| **Authentication & RBAC** | Completed | NextAuth Hybrid credentials & Azure AD | `/api/auth/*` | Secure middleware routing, customized command center login portal | Restricted access mobile shell, credentials login bypass | Route guard blocks, supervisors/admin access segregation checks |
| **SAP SuccessFactors Hub (Phase 5A)** | Completed | `SapConnection`, `SapSyncJob`, `SapSyncLog`, `SapFieldMapping`, `SapRetryQueue` | `/api/v1/sap/connections`, `/api/v1/sap/sync`, `/api/v1/sap/jobs`, `/api/v1/sap/logs`, `/api/v1/sap/mappings`, `/api/v1/sap/retry` | Connection health status, manual sync triggers, mapping manager editor, retry DLQ console, sync jobs list & logs dialog | No mobile changes required | Mock inbound employee sync deactivations, cost centers departments mapping, DLQ isolation, retry processes |
| **SAP Outbound Sync (5B.1 & 5B.2)** | Completed | `SapExportQueue`, `SapPayrollStage`, `SapReconciliationLog`, `SapPayrollPeriodLock` | `/api/v1/sap/export`, `/api/v1/sap/export/retry`, `/api/v1/sap/payroll`, `/api/v1/sap/reconciliation` | Outbound export queue list (idempotency keys, payloads), Reconciliation Board (discrepancy reports), Payroll Staging tab (freeze months, line approvals) | No mobile changes required | Leave/Attendance OData payloads, wage component mapping (`WT_OT_WKD`), staging freeze locked periods |
| **Reporting & Backups (6.1)** | Completed | `SavedReport`, `ReportExportLog`, `UserActivityLog`, `ProductionCheckLog`, `BackupJob`, `BackupAuditLog` | `/api/v1/reports/*`, `/api/v1/audit/logs`, `/api/v1/admin/backups/*` | Admin Backup Manager console (job run triggers, database status, export history grid, backup download/delete) | No mobile changes required (Self-service reports in 6.3) | Executive summary, CSV/JSON exports with security logs, password/AD secret exclusions, backup SHA-256 checksums |
